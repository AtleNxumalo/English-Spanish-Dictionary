# 📖 English-Spanish Dictionary

## Overview

The **English-Spanish Dictionary** is a simple C# console application that demonstrates the use of the `Dictionary<TKey, TValue>` collection. Users can enter an English word and receive its Spanish translation if it exists in the dictionary.

This project demonstrates dictionary operations, user input handling, loops, and conditional statements.

---

## Features

- Translate English words into Spanish
- Uses a `Dictionary<string, string>`
- Case-insensitive word lookup
- Interactive console interface
- Graceful handling of unknown words
